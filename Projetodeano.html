Mind Care

Sobre o que se trata o projeto

Nosso projeto tinha como objetivo a construção de dois ambientes, um site para publicação de vídeos, pesquisas, notícias que falavam sobre saúde mental; e um ambiente de conversa para que pessoas que passam por esses problemas poderem ter um ambiente aberto para conversar sobre.

Como foi fazer este projeto

Foi uma boa experiência, utilizamos html e css para produção do site e criamos um servidor no discord, duas coisas que são relativamentes fáceis, porém tivemos algumas dificuldades e não conseguimos divulgar esse projeto e fazer com que pessoas participassem.
